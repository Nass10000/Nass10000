* Desarrollo de un pipeline ETL automatizado con Python.
* Creación de más de 10 métricas y KPIs de logística, costos, ingresos y rentabilidad.

### 🤖 Predicción de abandono de clientes — Machine Learning

**Proyecto de clasificación para identificar clientes con riesgo de abandonar un banco**

* Análisis y preparación de un dataset de clientes bancarios.
* Limpieza, imputación y transformación de variables numéricas y categóricas.
* Creación de pipelines de preprocesamiento con Scikit-learn.
* Entrenamiento y comparación de modelos de clasificación.
* Uso de modelos como regresión logística y Gradient Boosting.
* Evaluación mediante accuracy, precision, recall, F1-score y ROC-AUC.
* Análisis de matrices de confusión y selección de umbrales.
* Evaluación del costo de falsos positivos y falsos negativos.
* **Herramientas:** Python, Pandas, NumPy, Scikit-learn y Jupyter Notebook.
